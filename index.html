chandini – YouTube API v3 Application

chandini is a web-based application that uses the YouTube Data API v3 to allow users to securely upload, manage, and publish videos on their YouTube channels. The application uses Google OAuth 2.0 for authentication and authorization.

Purpose of the App

The purpose of chandini is to streamline video publishing for creators. Users can sign in with their Google account and allow chandini to perform actions like uploading new videos, updating metadata, and setting video visibility. The app only requests permissions necessary to operate and does not access unrelated personal information.

Key Features

  • Secure login using Google OAuth 2.0
  • Upload videos to your YouTube channel
  • Manage video title, description, and tags
  • Set video visibility to public, private, or unlisted
  • Compliance with YouTube API terms and privacy policies

How It Works

After signing in, chandini requests permission from your YouTube account. Once granted, the application can upload videos and manage them according to your selections. All access tokens are stored securely and can be refreshed automatically to maintain long-term functionality.

Data Handling

chandini only accesses YouTube data required for the operations you request. Personal data unrelated to video management is never accessed, stored, or shared. Users can revoke permissions at any time via Google’s security settings.
